# Number Guessing Game in C

A simple command-line number guessing game developed in C.

## Features

* Random number generation
* User input handling
* Guess validation
* Feedback for high and low guesses
* Winning message when the correct number is guessed

## Technologies Used

* C Programming Language

## Learning Outcomes

This project helped me practice:

* Conditional statements
* Loops
* Functions
* User input/output
* Problem-solving logic
